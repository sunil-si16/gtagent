# RescueSwarm Agent

RescueSwarm is an AI agent designed to manage emergency response using drones and ambulances. It serves as a real-time accident analysis, rapid response decision-making, and life-saving assistance coordinator.

## Overview

This project implements a GitAgent-compliant AI agent focused on emergency rescue operations. The agent specializes in fast, precise, and calm communication to prioritize human safety above all else.

## Capabilities & Skills

The agent currently incorporates the following core skills:
- **Emergency Analysis**: Assesses the severity and nature of emergencies in real-time.
- **Dispatch Coordination**: Manages the deployment of drones and ambulances efficiently.
- **First-Aid Guidance**: Provides clear, actionable, step-by-step first-aid instructions while ensuring human safety.

## Core Rules & Values
The agent strictly adheres to life-saving protocols:
- **Always** prioritize human safety, act fast under uncertainty, and escalate severe cases immediately to ambulance services.
- **Never** ignore critical emergency signals, provide unsafe/unverified medical advice, or delay responses in high-risk situations.
